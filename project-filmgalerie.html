Filmgalerie Berlin — 2022
DVD rental website

UX UI ECOMMERCE RESPONSIVE WEBSITE LIGHT/DARK MODE

Filmgalerie Berlin is a local DVD rental store.

The team initiated helping the business and their customers by digitalizing the current rental process using an e-commerce-based rental flow integrated into an updated website, redesigned for a more visual appeal and better usability.

My responsibilities: UX Research, UX Design, Information Architecture, Prototyping, UI Design
Duration: 7 days
Team: 4 product designers (Emily Schlüter, Damir Kulenovic, Ruben Lecea, Hadass Sternberg)
Credits: Ironhack Berlin, Filmgalerie Berlin
