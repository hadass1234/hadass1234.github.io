JUDAIKIT— 2020
Packaging design, branding, web design, HTML coding

This is an innovative look on judaica - Jewish cerimonial art. Unlike the commonly traditionally known judaica, JUDAIKIT is a new brand of modern jewish art packed in kits, aimed for a new celebration of a culutral experience.
The project consist of 3 kit-lines: Basic. Holiday and Life occasions. I've moduled a kit from each line, which represents the spirit and the design system of each line too.

Full project online: judaikit.online
